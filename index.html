Python

Regarded as the best programming language for beginners. Easiest to learn. Widely used in scientific, technical and academic field, i.e. artificial intelligence. Django is a popular Python web framework.

C#

A popular choice for enterprises to create websites and Windows applications using the .NET framework. ASP.NET is the web framework from Microsoft. Similar to Java in basic syntax and some features.

Java

Very popular on all platforms, OS, and devices due to its portability. One of the most in-demand and highest paying programming languages.

C++

Complex version of C with a lot more features. Widely used for developing games, industrial and performance-critical applications. Learning C++ is like learning how to manufacture, assemble and drive a car. Recommended only if you have a mentor to guide you.

Objective-C

The primary language that Apple uses for Mac OS X and iOS. Choose this if you want to focus on developing cross-platform Apple apps only. Consider learning swift as your next programming language.

C

One of the oldest and most widely used languages in the world. Popular for system and hardware programming. A subset of C++.

JavaScript

The most popular client-side web scripting language. A must-learn for fron-end web developers (plus HTML & CSS). One of the hottest programming languages at the moment, due to its increasing server-side application (Node.js)

Ruby

Mostly known for its popular web framework - Ruby On Rails. Designed for fun & productive coding, with a focus on getting things done. Best for fun, personal projects, startups and/or rapid development.

PHP

This is a server-side scripting language designed primarily for web development. A powerful tool for dynamic and interactive web-pages. PHP code may be embedded into HTML code, or it can be used in combination with various web template systems, web content management systems and web frameworks.
